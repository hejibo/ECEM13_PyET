Exercise 1.8 - formatted strings
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Given these variables only::

    a = 'Hello'
    name = 'Jon'
    height = 1.80
    hour = 9
    minute = 5

use string formatting to print out the following:

    Hello Jon
    HELLO Jon
    The time is 9.05am
    The time is 09:05
    Jon is 1.8m tall
